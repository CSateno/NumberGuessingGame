# NumberGuessingGame
My very first coding project in C++ using loops and conditional statements. 

# Features
- Easy mode: Unlimited guesses with high and low hints.
- Hard mode: Only 3 attempts to guess the number correctly.

# How to compile and run
g++ numberguessing.cpp -o game
./game

# What I learned:
-  Switch statements and do-while loops.
-  Input validation.
